Plugin

Bring shapefly to your life.Lets users drawing ready-made shapes on your website straight-through.

Shapefly plugin on WYSIWYG editor

Suppose you are using the following editor on your website.You want to insert a flowchart here right now.Then you click the button on the toolbar above, then it will redirect to shapefly.com service for you to draw something and insert to the editor after finish.

How it work

As you know from the sample above, the plugin just redirect you to open shapefly.com/editor, all your drawing was on shapefly.com service, and it just return a image to your website after finish.Furthermore, if you want to change your drawing, you can select the image, then also click the plugin button.It redirects you to shapefly.com/editor for changing your drawing of the image.So don't forget to save your drawing on shapefly service after finish.

How to use

The following is the plugins for different scenarios, you can select a suitable one.

Independent javascript

The shapefly-plugin is a independent javascript liberary, which only includes two files shapefly-plugin.js and shapefly-plugin.html.For the security of cross-domain, you have to copy them to your owner website before using it.Then the following is really all you need to get going.

<script src="js/shapefly-plugin.js"></script>
<script>
  var btn = document.getElementById('sfBtn');
  shapefly.plugin(btn, {
    finish: function(img) {	  
      //TODO:insert the image to your website
	  document.getElementById('imgDiv').innerHTML = img;
	}
  });
</script>
...

<input type="button" id="sfBtn" value="Shapefly" />

Or install with bower

bower install shapefly-plugin

Plugin for tinyMCE

The tinymce-plugin-shapefly is a tinyMCE plugin, which is a button on tinyMCE toolbar.Config the tinymce simply with the name shapefly.It is built base on the shapefly-plugin.See the following to learn about the config on tinyMCE.

<script src="shapefly-plugin.js"></script>
<script src="tinymce-plugin-shapefly.js"></script>
	  tinymce.init({
		selector: "textarea",
		plugins: [
			"advlist autolink lists link image charmap print preview anchor",
			"searchreplace visualblocks code fullscreen",
			"insertdatetime media table contextmenu paste imagetools shapefly"
		],
		toolbar: "insertfile undo redo | styleselect | bold italic | alignleft aligncenter alignright alignjustify | bullist numlist | link image shapefly",
		autosave_ask_before_unload: false,
		max_height: 200,
		min_height: 160,
		height : 180
	  });
</script>
...

<textarea id="tinymceDemo"></textarea>

Or install with bower

bower install tinymce-plugin-shapefly
