插件

怎样使用Shapefly插件,让用户直截了当地在你的网站上插入图形或流程图.让Shapefly为你的网络生活服务.

在线编辑器中使用Shapefly

假设你正在使用下面的在线编辑器,而此时你想插入一张流程图,你会怎么办?Shapefly插件让你只需点击上面的工具条中的按钮, 就可以访问Shapefly服务,画任何你想画的东西.最后,点击完成就可以在编辑器中插一张你所画的图片.

工作原理

就象上面的实例,shapefly插件帮你打开shapefly.com/editor, 然后以图片的形式返回到你的网站, 一切就是这么地直截了当.甚至你还可以编辑你所画的图形,只需在编辑器中选中,点击上述的按钮,就可以打开Shapefly服务进行编辑. 因为所有在Shapefly中所画的图形或流程图都会有一个唯一的ID,所以当你在Shapefly完成你所画的,别忘了保存,以便日后使用或编辑!

怎么使用

我们为不同的应用场景,提供了便于使用的插件,日后还会提供更多合适的插件.请选择下面适合你的插件.

无依赖纯javacript插件

jshapefly-plugin是一个无依赖,独立的javascript插件, 它只包含两个文件shapefly-plugin.js和shapefly-plugin.html.由于浏览器的同源策略问题, 使用前请将这两个文件放入你的网站同一目录下.然后,下面简单的代码就是你要做的:

<script src="js/shapefly-plugin.js"></script>
<script>
  var btn = document.getElementById('sfBtn');
  shapefly.plugin(btn, {
    finish: function(img) {	  
      //TODO:insert the image to your website
	  document.getElementById('imgDiv').innerHTML = img;
	}
  });
</script>
...

<input type="button" id="sfBtn" value="Shapefly" />

使用bower安装

bower install shapefly-plugin

tinyMCE插件

tinymce-plugin-shapefly是为tinyMCE提供的插件, 也是shapefly-plugin的一个扩展.该插件通过简单配置名称为shapefly, 就能在tinyMCE工具条出现一个shapefly按扭.配置示例:

<script src="shapefly-plugin.js"></script>
<script src="tinymce-plugin-shapefly.js"></script>
	  tinymce.init({
		selector: "textarea",
		plugins: [
			"advlist autolink lists link image charmap print preview anchor",
			"searchreplace visualblocks code fullscreen",
			"insertdatetime media table contextmenu paste imagetools shapefly"
		],
		toolbar: "insertfile undo redo | styleselect | bold italic | alignleft aligncenter alignright alignjustify | bullist numlist | link image shapefly",
		autosave_ask_before_unload: false,
		max_height: 200,
		min_height: 160,
		height : 180
	  });
</script>
...

<textarea id="tinymceDemo"></textarea>

使用bower安装

bower install tinymce-plugin-shapefly

WordPress插件

wp-tinymce-plugin-shapefly是一个WordPress插件, 安装该插件能让WordPress编辑器在发布博客时通过在线访问shapefly服务,直截了当地插入图形或流程图.点击链接进行下载, 然后安装到你的WordPress.
